XE Author List Widget Plugin displays list of authors. It also displays authors Email avatars.


XE Author List Widget Plugin displays list of authors. It allows to admin number of authors to display. admin can set author avatars image size email.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `xe-author-list-widget` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Head over the the widgets settings page and enable `XE Author List Widget`

== Frequently Asked Questions ==

= How to use it? =
